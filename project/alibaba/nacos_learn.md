## 安装
在[nacos-github](https://github.com/alibaba/nacos/releases)找到安装文件,解压安装.
```shell script
wget https://github.com/alibaba/nacos/releases/download/1.1.4/nacos-server-1.1.4.tar.gz
tar -zxvf nacos-server-1.1.4.tar.gz 
```
